Linear Regression

Linear Regression is a very easy and common tool to perform Data Analysis and predictions on a set of data.
This page showcases a simple example built from scratch, consisting of deducing the weight of a person, given its height.

A virtual dataset of people measurements has been created and will be used throughout the rest of the page as our working tool.
